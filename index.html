My Web Background

Who is Philip Tuan?

Well, I'm glad you asked! In the 7th grade, I was introduced to the web when I created my first website in the windows default notepad program using only html. It wasn't pretty, but to me it was the most amazing thing I had ever done in my life! Wanting to make websites with better design and more interactive, I started learning CSS then JavaScript on codecademy. Finishing the codecademy courses wasn't enough for me so I joined codeschool to polish up my programming skills. However, I wanted to improve my web development skills further plus create clean concise code. I achieved all I wanted when I discovered CareerFoundry Certification Web Development Course. At the same time, I was taking a supplement course, FreeCodeCamp Full Stack Web Development Course to brush up on what I learned in the CareerFoundry course. In every job I had, my bosses all told me the same thing, that I was a hard worker who was always punctual. Always determined to finish the job in the most efficient way possible.
